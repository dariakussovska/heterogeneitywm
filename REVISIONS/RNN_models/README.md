### RNN model files

The `.mat` RNN files are not included in this GitHub repository because they exceed GitHub's file-size limits.

To run the RNN analyses, download the model files from OSF:

https://osf.io/md4wg

After downloading, place the `.mat` files in the following folder:

```text
REVISIONS/RNN_models/
```
Without this, you won't be able to run the 
```text
RNN_decoding_comparison.py
```
 file.
